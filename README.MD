# ALEXBOT V3 PRO (Experimental Visual Predictor)

Proyecto experimental de visiГіn por computador para estimar el ГЎngulo de una bola y el marcador de referencia del rotor dentro de una ROI calibrada.

## Estructura


## Flujo de ejecuciГіn

1. Ejecuta `calibrate.py` para generar `config.json` con la ROI.
2. Ejecuta `main.py`.
3. El sistema:
   - captura video de la ROI,
   - detecta bola y marcador verde del rotor,
   - estima cinemГЎtica (omega/alpha) para la bola,
   - proyecta ГЎngulo relativo bola-vs-rotor,
   - muestra el resultado en un overlay.

## Modelo fГӯsico actual (resumen)

1. Estima `Пү` instantГЎnea con diferencias angulares envueltas.
2. Ajusta una regresiГіn local `Пү(t)` para derivar `Оұ` efectiva.
3. Identifica coeficientes de fricciГіn lineal + Coulomb.
4. Integra la dinГЎmica hasta umbral de caГӯda para obtener `t_drop`.
5. Proyecta ГЎngulo de impacto y aplica una dispersiГіn heurГӯstica por deflectores.
